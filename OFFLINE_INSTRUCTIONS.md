# Автономный запуск PhysLab AI v3

## Проверка

```bash
python check_architecture.py
python check_all_models.py
```

## Запуск интерфейса

```bash
python -m streamlit run app_backup.py
```

## Полное повторное обучение

```bash
python build_all.py
```

Команда пересоздает датасеты, модели, таблицы метрик, матрицы ошибок и диаграммы важности признаков для всех четырех лабораторных.

## Реальные CSV

Шаблоны находятся в папке `real_data_templates`.

В интерфейсе выберите лабораторную, затем `Загрузить CSV`. Распространенные названия столбцов будут сопоставлены автоматически.
