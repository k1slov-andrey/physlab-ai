
# PhysLab AI

**Интеллектуальная диагностика ошибок цифрового физического эксперимента**

PhysLab AI — исследовательский прототип для анализа временных рядов, получаемых при изучении закона охлаждения Ньютона. Система аппроксимирует физическую модель по измеренным данным, извлекает интерпретируемые признаки и определяет вероятный тип ошибки с помощью машинного обучения.

Проект разработан для участия в **JUNIOR ML Contest 2026** и поступления в магистратуру ИТМО «Искусственный интеллект».

---

## Проблема

Цифровые лаборатории позволяют автоматически регистрировать температуру и строить графики, однако сами по себе не объясняют:

- является ли эксперимент корректным;
- чем вызвано отклонение;
- на каком этапе возникла ошибка;
- что учащемуся или преподавателю следует проверить.

При работе с группой учащихся преподавателю приходится вручную просматривать большое количество временных рядов. Такой анализ требует времени и зависит от опыта специалиста.

ML-классификация
        ↓
Диагноз, вероятность и педагогическая рекомендация
