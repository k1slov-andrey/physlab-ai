# PhysLab AI

Интеллектуальная система диагностики ошибок цифрового физического эксперимента.

Проект является технологическим продолжением выпускной квалификационной работы, посвященной использованию цифровых лабораторий при изучении молекулярной физики и термодинамики.

## Цель проекта

Разработать систему, которая анализирует данные температурного датчика, обнаруживает отклонения от ожидаемой физической модели и формирует образовательную подсказку учащемуся.

## Текущий этап

- создан репозиторий проекта;
- настроено виртуальное окружение Python;
- формируется генератор синтетических данных;
- разрабатывается классификация ошибок эксперимента.
